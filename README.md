# Voting-App
NodeJS backend for voting app + React on front

## Main Technologies

* [x] **JavaScript ES5/ES6+**
* [x] **NodeJS**
* [x] **Express**
* [x] **MongoDB**
* [x] **React/Redux**
* [x] **NPM**
* [x] **ChartJS**

### Setup

```bash
$ git clone or download
```

Go to project direction

Install dependencies for server

```bash
$ npm install

```

Install dependencies for client

```bash
$ cd client
$ npm install

```
Run server (nodemon)

```bash
$ nodemon

```

Run server (client)

```bash
$ cd client
$ npm start

```
